Drag a task card into any day. Drop it, tick it off, and jot a quick note about what you covered. Click curriculum on a task to track your place in the course itself, and use the ✎ / 🗑 icons to edit or remove tasks and sub-items.
Data is saved to this browser only (localStorage). Use "Export backup" occasionally so your streak survives clearing cache or switching devices — or set up GitHub sync above to keep every device in step automatically.

Cloud sync via GitHub

Store your data as a JSON file in a GitHub repo so every laptop you open this page on stays in sync. Create a fine-grained personal access token scoped to just this one repo with "Contents: Read and write" permission. The token is saved only in this browser's local storage — never uploaded anywhere except directly to GitHub's API.
